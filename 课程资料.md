## 安装 PySide6 amzqr

1. 国外源

pip install pyside6   
pip install amzqr

2. 国内源（如果国外源安装太慢，可使用国内源进行安装）

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pyside6  
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple amzqr
